# 페페의모험 (Pepe's Adventure)

브라우저에서 바로 즐기는 고전 스타일 RPG 웹게임입니다.

**▶ 플레이하기: https://shinpepe.github.io/pepes-adventure/**

## 게임 소개

몬스터를 사냥해 골드와 경험치를 모으고, 장비를 갖춰 네 지역의 보스를
차례로 격파하는 클래식 RPG입니다.

- 4개 지역과 몬스터, 지역 보스
- 상점 · 여관 · 식당 · 도박장이 있는 마을
- 훈장, 칭호, NPC 이벤트
- 3슬롯 저장 (브라우저에 저장됩니다)

## 조작법

| 키 | 동작 |
|---|---|
| 마우스 | 모든 버튼 조작 |
| SPACE / ENTER | 진행 · 확인 |
| Z | 공격 |
| S | 스킬 |
| X | 도망 |
| ESC | 뒤로 · 시스템 메뉴 |
| M | 지도 |
| B | 훈장 보관함 |
| R | (게임 오버 시) 부활 |
